# Indian Stock Market Prediction using Machine Learning

This project predicts stock prices using historical data from NSE.

Features:
- Data collection using yfinance
- Data visualization using matplotlib
- Feature engineering (Moving Averages)
- Linear Regression model
- Streamlit web app for prediction

Tech stack:
Python, Pandas, Scikit-learn, Streamlit, Matplotlib, yfinance

Run project:

pip install -r requirements.txt
streamlit run app.py
